# Gold Tokenization Platform Landing Page

## Overview
A landing page for a gold tokenization platform that showcases the process of converting physical gold into blockchain-based digital tokens and enables users to borrow against their tokenized gold holdings. The platform includes an articles section with tokenization insights and educational content.

## Navigation
- Two-part navigation system with "Home" and "Articles" menu options
- Dynamic navigation between sections with smooth scrolling
- "Back to Home" functionality available from Articles section
- Responsive navigation design for all devices
